# Merca-Todo 🍲🧀🌮
***
Proyecto Colaborativo sobre un inventario online de un supermercado. Utilizaremos React, Django y MySql.

## Technologies 🛠️
* **React**
* **Mysql2**
* **Django**
* **Bootstrap**
* **CSS3**

***

## Installation 📝
***

$ git clone https://github.com/Fernaandaa/mercatodo.git
$ cd mercatodo/mercatodo 
$ npm i  / npm install
$ npm start
